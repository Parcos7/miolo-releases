# Miolo

Bem-vindo ao repositório oficial de downloads do **Miolo**! 

> **O que é o Miolo?**
> O Miolo é um aplicativo desenvolvido para gerenciar vendar de uma cafeteria. Ele foi criado com foco em ser simples, rápido e direto ao ponto.

---

## Como Baixar e Usar

O programa não requer uma instalação complexa. Ele é um único arquivo executável pronto para uso.

1. Acesse a aba **[Releases (Lançamentos)] aqui no GitHub.
2. Baixe o arquivo mais recente chamado `Miolo.exe`.
3. Coloque o arquivo em uma pasta da sua preferência no seu computador.
4. Dê um duplo clique para abrir e pronto!

## Atualizações Automáticas

Você não precisa se preocupar em voltar aqui no site toda vez que uma versão nova sair. O Miolo possui um **atualizador inteligente integrado**. 

Sempre que eu lançar uma melhoria ou correção, o próprio programa vai te avisar na tela e fazer a substituição de forma automática e segura para você.

## Nota sobre Antivírus (Falsos Positivos)

Como este é um software independente (indie) compactado em um único arquivo `.exe`, é comum que o Windows Defender (ou o recurso SmartScreen) o bloqueie na primeira execução por não reconhecer o desenvolvedor. Isso é conhecido como "falso positivo". 

Se a tela azul do Windows aparecer, basta clicar em **"Mais informações"** e depois em **"Executar assim mesmo"**. O sistema é 100% seguro!
